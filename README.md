# 🧮 BMI Calculator App (Flutter)

A simple and elegant **Body Mass Index (BMI) Calculator** built using **Flutter**.
This app helps users calculate their BMI based on height and weight and provides instant health classification.

---

## 📱 Features

* ⚡ Fast BMI calculation
* 📊 Instant result display
* 🧍 Height & weight input
* 🏷️ BMI category classification (Underweight, Normal, Overweight, Obese)
* 🎨 Clean and responsive UI
* 📱 Works on Android & iOS

---

## 🧮 BMI Formula

BMI is calculated using:

```
BMI = weight (kg) / height² (m²)
```

---

## 🏥 BMI Categories

| BMI Range   | Category      |
| ----------- | ------------- |
| < 18.5      | Underweight   |
| 18.5 - 24.9 | Normal weight |
| 25 - 29.9   | Overweight    |
| 30+         | Obese         |

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/vishal-rajpoot-1530/Projects/bmi-app.git
```

### 2. Navigate to project directory

```bash
cd bmi-app
```

### 3. Install dependencies

```bash
flutter pub get
```

### 4. Run the app

```bash
flutter run
```
